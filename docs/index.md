# Имитационное моделирование -- Конспект курса

Конспект покрывает все ключевые темы курса: от общего введения в имитационное моделирование до математического аппарата теории массового обслуживания и практического программирования симуляций на Python (SimPy).

---

## Как пользоваться

- **Последовательное чтение**: темы выстроены от общего к частному -- начните с введения
- **Справочник**: используйте [шпаргалку](cheatsheet.md) для быстрого поиска формул
- **Поиск**: используйте строку поиска вверху страницы

!!! tip "Совет"
    Каждая тема начинается с блока **"Зачем это нужно"** -- если времени мало, прочитайте хотя бы его.
